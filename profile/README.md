## What we build

We ship AI tools that run on your laptop, not someone else's GPU cluster. Four areas:

- **🔍 Vision** — annotation tools, edge detection & segmentation kits
- **🤖 Assistants** — on-device multimodal & voice agents for desktops and robots
- **⚙️ Platform** — training pipelines and self-hosted inference
- **📚 Applied** — real-world AI products, including language learning
